# 《灿烂》

> 一部关于成长、自我认同与生命状态的原创动画电影

---

## 项目简介（Project Overview）

《灿烂》是一部约15~20分钟的原创动画电影。

故事来源于创作者真实的人生经历，但它并不是一部传统意义上的自传。

它没有反派，没有逆袭，没有刻意制造戏剧冲突。

它只是希望回答一个问题：

> **如果人生不同阶段的自己能够相遇，他们会怎么看待彼此？**

电影采用平行时间的表现形式。

13岁的自己、17岁的自己、20岁的自己、24岁的自己以及现在的自己，同时生活在同一部电影中。

他们不会互相改变。

他们只是互相看见。

最后共同证明一句话：

> **我没有变成不同的人。**
>
> **我只是在人生不同的阶段，让不同的一部分自己站到了最前面。**

---

# 创作目标

不是制作一部短视频。

不是制作一部励志电影。

而是制作一部能够被反复观看的动画短片。

希望观众看完以后记住的不是剧情。

而是重新开始理解自己。

# 表现形式

二维动画电影

现实主义场景

轻微幻想设定

无旁白

极少对白

环境音驱动叙事

大量留白

---

# 美术风格

整体参考：

宫崎骏电影的生活感

而不是模仿吉卜力画风。

遵循：

真实建筑

真实城市

真实生活

真实人物

但是拥有：

柔和空气

自然风

阳光

植物

光影

以及一点点不可解释的魔法。

---

# 核心创作原则

整个电影遵守唯一原则：

**不要描写发生了什么。**

**要描写什么一直没有改变。**

所有剧情、分镜、美术、音乐，都必须服务于这一原则。
